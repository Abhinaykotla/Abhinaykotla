## Image Specifications

**Recommended Settings:**

- **Aspect Ratio:** 16:9 (landscape) or 4:3
- **Resolution:** Minimum 1920x1080px for web use
- **Format:** JPG or PNG (PNG for transparency if needed)
- **Style Keywords:** professional, clean, modern, tech-focused, portfolio-quality, minimalist

**Recommended AI Tools:**

1. **DALL-E 3** (via ChatGPT Plus or API) - Best for detailed technical illustrations
2. **Midjourney** - Excellent for stylized, professional imagery
3. **Stable Diffusion** (via Automatic1111 or online services) - Free, highly customizable
4. **Adobe Firefly** - Good for commercial use without copyright concerns
5. **Leonardo.ai** - Great for consistent style across multiple images

**Pro Tips:**

- Add style modifiers: "professional tech illustration, clean design, suitable for portfolio website"
- For consistency: Generate all images in one session with similar style parameters
- Use negative prompts: "no text, no watermarks, no cluttered elements"
- Consider brand colors if you have a specific color scheme

## Blog Image Specifications

**Recommended Settings for Blog Images:**

- **Aspect Ratio:** 21:9 or 16:9 (wide banner for hero images)
- **Resolution:** 2560x1080px (21:9) or 1920x1080px (16:9) minimum
- **Format:** JPG (for photographs/complex images) or PNG (for graphics with transparency)
- **File Size:** Optimize to under 500KB for web performance
- **Style Keywords:** "blog header, hero image, professional, clean, tech-focused, educational, modern"

**Blog-Specific Tips:**

- **Hero images** should be wider and more dramatic than project thumbnails
- Use **text-safe areas**: keep important visuals centered or to the left, as blog titles often overlay the image
- Consider **dark overlays** or gradient overlays that make white text readable
- Maintain **consistent style** across all blog posts for brand cohesion
- For tutorial/guide posts, show **code or UI elements** clearly
- For research/technical posts, include **diagrams or visualizations**
